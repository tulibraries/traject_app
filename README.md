# TU Traject App

Stand alone traject app based on [traject/traject](https://github.com/traject/trajec://github.com/traject/traject)


To execute

```bash
git clone https://github.com/tulibraries/traject_app.git
cd traject_app
bundle install
solr_wrapper &
traject -c settings.rb JB-Export-1.out
```
